Scaling Summary

All these fonts scale identically because they are:

  • Vector-based (like SVG) — no pixelation at any size
  • Web fonts via Google Fonts — render at browser resolution
  • Variable-weight capable — smooth transitions from 400→700 weight
  • Responsive — automatically adapt to mobile, tablet, desktop without loss

Change font-family in CSS → all sizes update instantly, no rendering differences.
